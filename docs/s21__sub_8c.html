Subtracts one s21_decimal number from another.

This function subtracts the provided s21_decimal number value_2 from value_1 and stores the result in result. The subtraction operation is implemented by negating the sign of value_2 and then performing addition using the s21_add function.

Parameters
value_1The s21_decimal minuend (number from which another is to be subtracted).
value_2The s21_decimal subtrahend (number that is to be subtracted).
resultPointer to the s21_decimal structure where the subtraction result will be stored.

Returns a status indicating the success or error of the operation.
