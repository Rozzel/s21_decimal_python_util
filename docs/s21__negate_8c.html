Negates an s21_decimal value.

This function inverts the sign of a given s21_decimal number. If the number is positive, it will be turned negative and vice versa. Numbers with zero value remain unchanged.

Parameters
valueThe s21_decimal number to be negated.
resultPointer to the s21_decimal variable where the result will be stored.
Returns
OTHERS_OK if the operation is successful, OTHERS_CALCULATION_ERROR if result is NULL.
Note
  • The function checks the sign bit of the provided value and flips it to negate the number.
  • This function does not change the magnitude or absolute value of the number, only its sign.
