Converts a s21_decimal number to a float representation.

This function takes a given s21_decimal number (src) and converts it to its float representation, storing the result in the provided dst pointer. The conversion uses an optimized method by breaking down the number using precomputed exponentiation values and accumulating the result in a double precision floating point. This double result is then converted to float before returning.

Parameters
srcThe s21_decimal number to be converted.
dstPointer to store the converted float value. The function assumes that the caller provides a valid memory location.
Returns
Returns CONV_OK upon successful conversion, and CONV_ERROR if dst is NULL.
Note
This function assumes the scaling factor in the s21_big_decimal structure is an index into the fastExps array, so it should not exceed the size of the array.
