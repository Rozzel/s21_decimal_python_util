enum UtilCompare

Enumerates the possible outcomes of a comparison operation.

Represents the three potential outcomes when comparing two values.

  • CMP_EQUAL: Both values are equal.
  • CMP_LESS: The first value is less than the second value.
  • CMP_GREATER: The first value is greater than the second value.
Enumerator
CMP_EQUAL 

Represents a situation where two values being compared are equal.

CMP_LESS 

Represents a situation where the first value is less than the second one.

CMP_GREATER 

Represents a situation where the first value is greater than the second one.

◆ banker_rounding_big_decimal()

void banker_rounding_big_decimal ( s21_big_decimal *  bigValue)

Applies bankers' rounding to an s21_big_decimal number.

Bankers' rounding, also known as unbiased or convergent rounding, rounds a number to the nearest even number when the number is halfway between two numbers. For instance, 0.5 rounds to 0 and 1.5 rounds to 2. This method reduces the accumulation of rounding errors.

The function iteratively rounds the least significant digits of the number based on the bankers' rounding rules until the desired precision (scaling factor) is reached or the least significant digits become zero.

Parameters
bigValuePointer to the s21_big_decimal object to be rounded using bankers' rounding.
Note
Ensure that the bigValue pointer is not NULL before calling this function. The function does not check for NULL pointers and assumes that a valid pointer is provided.
Warning
This function alters the original value of the s21_big_decimal object to reflect the rounded value.

Obtains the last count digits of the mantissa of an s21_big_decimal number.

This function calculates the value of the last count digits of the mantissa of the given s21_big_decimal number. It first creates a copy of the original number, scales it down by dividing it by 10 count times and then scales it back up by multiplying it by 10 count times. The difference between the original number and the scaled number gives the last count digits of the mantissa, which are then returned after calculating the modulus with 10 raised to the power of count.

Parameters
minuendPointer to the s21_big_decimal number whose last count digits of the mantissa are to be calculated.
countThe number of digits from the last to be calculated.
Returns
The value of the last count digits of the mantissa of the s21_big_decimal number.
Note
The function assumes that minuend is a valid pointer to an s21_big_decimal object and does not check for NULL pointers.
Warning
The function may produce undefined results if count is negative or greater than the total number of digits in the mantissa.
