Rounds an s21_decimal value to the nearest integer.

This function rounds a given s21_decimal number to the nearest integer based on its fractional part. Numbers with a fractional part equal to or greater than 0.5 are rounded up, while others are rounded down. The behavior follows the traditional mathematical rounding rules.

Parameters
valueThe s21_decimal number to be rounded.
resultPointer to the s21_decimal variable where the result will be stored.
Returns
OTHERS_OK if the operation is successful, OTHERS_CALCULATION_ERROR if result is NULL.
Note
  • This function separates the integer and fractional parts of the number.
  • If the fractional part is 0.5 or greater, the integer part is incremented.
  • For negative numbers, the number is decreased instead.
