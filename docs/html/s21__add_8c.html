Adds two s21_decimal values.

This function calculates the sum of two s21_decimal values. It first converts the s21_decimal values to a big decimal representation. After performing the addition (or subtraction, if one of the numbers is negative), it converts the result back to s21_decimal and applies rounding.

Parameters
value1First operand of the addition.
value2Second operand of the addition.
resultPointer to a variable where the result will be stored.
Returns
Status of the addition operation. If the operation is successful, the function returns 0. Non-zero values indicate errors.
