Divides two s21_decimal values.

This function divides value_1 by value_2 using a "big decimal" representation to handle precision and arithmetic intricacies. After performing the division, the result is converted back to the s21_decimal format with rounding applied as necessary. If value_2 is zero, division by zero error code is returned.

Parameters
value_1The dividend.
value_2The divisor. If this is zero, an error code is returned.
resultPointer to store the result of the division.
Returns
Status of the operation. Returns ARITHM_DIVISION_BY_ZERO if value_2 is zero. Returns 0 if successful, and non-zero values indicate other errors.
