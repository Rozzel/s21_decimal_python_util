Computes the floor of an s21_decimal value.

The function calculates the floor of the given s21_decimal number. The floor of a number is the largest whole number less than or equal to the number. If the number is already an integer, it remains unchanged. If the number is negative and has a fractional part, the resulting value is decremented by 1 to give the floor value.

Parameters
valueThe s21_decimal number whose floor is to be calculated.
[out]resultA pointer to the s21_decimal value that will store the floor of the given number.
Returns
OTHERS_OK on successful computation, and OTHERS_CALCULATION_ERROR in case of errors such as null pointer arguments.
Note
  • The function uses helper functions such as s21_truncate, is_zero_decimal, and s21_sub.
  • It also utilizes a constant NEGATIVE_SIGN_MASK to identify negative numbers in the s21_decimal format.
