Converts a s21_decimal number to an integer representation.

This function converts the given s21_decimal number (inputDecimal) to its integer representation. The conversion process first truncates the decimal number to ensure there are no fractional components. After truncation, the function checks if the number can fit within a 32-bit signed integer. If it can, it converts the decimal number into an integer and places it in the convertedInt pointer.

Parameters
inputDecimalThe s21_decimal number to be converted.
convertedIntPointer to store the converted integer value. The function assumes that the caller provides a valid memory location.
Returns
  • CONV_OK upon successful conversion.
  • CONV_ERROR if convertedInt is NULL or if the number cannot be safely converted to an integer.
Note
The function assumes that the s21_decimal value does not exceed the range of a 32-bit signed integer and that truncation will not introduce any rounding errors that could lead to inaccurate results.
