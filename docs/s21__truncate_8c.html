Truncates the fractional part of an s21_decimal value.

This function truncates, or discards, the fractional part of a given s21_decimal number, effectively converting the number to its integer representation.

Parameters
valueThe s21_decimal number whose fractional part will be truncated.
resultPointer to the s21_decimal variable where the result will be stored.
Returns
  • OTHERS_OK if the operation is successful.
  • OTHERS_CALCULATION_ERROR if result is NULL or the scale exceeds MAX_EXPONENT_SIZE.
Note
  • This function does not round the value; it merely removes the fractional part.
  • The sign of the original number is preserved in the result.
