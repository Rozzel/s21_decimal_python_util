Detailed Description

A decimal representation using an array of bits.

This structure is used for representing decimals in a specialized manner. Each element of the array 'bits' contributes to the overall decimal value.

unsigned int s21_decimal::bits[4]

The array of bits where each bit represents part of the decimal.

  • bits[0]: Contains the least significant 32 bits of the 96-bit integer part. This portion contributes the smallest value to the decimal number.
  • bits[1]: Contains the middle 32 bits of the 96-bit integer part. This portion contributes the middle value to the decimal number.
  • bits[2]: Contains the most significant 32 bits of the 96-bit integer part. This portion contributes the largest value to the decimal number.
  • bits[3]: Contains metadata including the sign and scaling factor.
    • Bits 0-15: Unused, must be zero.
    • Bits 16-23: Holds the exponent for the scaling factor, ranging from 0 to 28. This exponent is used as the power of 10 to scale the 96-bit integer.
    • Bits 24-30: Unused, must be zero.
    • Bit 31: Sign bit. 0 means the number is positive, and 1 means it's negative.
